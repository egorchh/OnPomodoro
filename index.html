  1. What is OnPromodoro?

    OnPomodoro is a customizable pomodoro timer that works on desktop & mobile browser. The aim of this app is to help you focus on any task you are working on, such as study, writing, or coding. This app is inspired by Pomodoro Technique which is a time management method developed by Francesco Cirillo.

  2. What is Promodoro Technique?

    The Pomodoro Technique is created by Francesco Cirillo for a more productive way to work and study. The technique uses a timer to break down work into intervals, traditionally 25 minutes in length, separated by short breaks. Each interval is known as a pomodoro, from the Italian word for 'tomato', after the tomato-shaped kitchen timer that Cirillo used as a university student - Wikipedia

  3. How to use Promodoro Timer?
    1. 1. Add tasks to work on today
    2. 2. Set estimate pomodoros (1 = 25min of work) for each tasks
    3. 3. Select a task to work on
    4. 4. Start timer and focus on the task for 25 minutes
    5. 5. Take a break for 5 minutes when the alarm ring
    6. 6. Iterate 3-5 until you finish the tasks
  4. Features
    1. 1. Responsive design that works with desktop and mobile
    2. 2. Color transition to switch moods between work time and rest time
    3. 3. Audio notification at the end of a timer period
    4. 4. Customizable timer intervals to suit your preference
